Cadastro de faturamento

Voltar

TUTORIAL passo a passo dentro da Retaguarda

No cliente desejado:

Aba "PARMETROS" > PARMETROS > selecione a opo "PERMITIR FATURAR" > Escolha o tipo de faturamento e o perodo > "SALVAR".

Nas seguintes situaes, pode marcar as outras opes, so elas:

  • Emitir NFP: Caso queira que o cliente obtenha Nota Fiscal Paulista

  • Gerar Boleto: Caso queira que um boleto seja gerado

  • Emitir NF-e: Caso queira que seja emitida nota fiscal eletrnica

  • KM Obrigatria: Quando esta opo marcada, obrigatrio inserir a quilometragem do cliente, caso contrrio no ser permitido faturar

  • Motorista Obrigatrio: Quando esta opo marcada, obrigatrio inserir o nome do motorista, caso contrrio no ser permitido faturar

  • OBSERVAO:

Caso o cliente no esteja cadastrado clique no boto abaixo para tutorial passo a passo de cadastro.
